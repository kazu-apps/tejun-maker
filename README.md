# 手順書メーカー

実務品質の作業手順書を、ブラウザだけで作成できる無料ツールです。

**▶ 使ってみる: https://kazu-apps.github.io/tejun-maker/**

## 特徴

- 🧭 **実務の型に沿った入力フォーム** — 事前確認 / 作業手順(期待結果つき) / 切り戻し(発動基準つき) / 正常性確認
- 📄 **そのまま提出できる紙面プレビュー** — 承認欄・チェックボックス付き。印刷/PDF出力対応
- 📝 **Markdown出力** — Confluence / Notion / GitLab / Redmine にそのまま貼れる
- 🔒 **完全ローカル処理** — 入力データはサーバに一切送信されません(保存はブラウザのlocalStorageのみ)
- 💾 JSON形式での保存・読み込みに対応

## 使い方

1. 左のフォームに作業内容を入力(まずは「サンプル」ボタンでイメージを掴むのがおすすめ)
2. 右のプレビューで仕上がりを確認
3. 「Markdownコピー」でWikiへ、「印刷/PDF」で提出用に出力
